# Install tutorial

You only need to clone this repo.

Then go to this repo root, then run:
```
pip install -e .
```

If you want to install plugins, run:
```
pip install -e .[<name plugin 1>, <name plugin 2>, ...]
```

Example:
```
pip install -e .[sanet]
```

List of current support:
- sanet
